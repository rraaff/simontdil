Desinstalacion

Para desinstalar simon, ya sea porque se quiere remover o se quiere reinstalar por algun error en la
 instalacion, se deben seguir los siguientes pasos.
 
 1) Ejecutar los siguientes comandos para eliminar el servicio de MySQL
	net stop MySQL
	sc delete MySQL
 2) Ir al Panel de Control >> Add Remove Programs y eliminar:
 	Java DB 10.5.3.0
 	Java(TM) 6 Update 21
 	Java(TM) SE Development Kit 6 Update 21
 	MySQL Server 5.5
 3) Eliminar el directorio de instalacion, por ejemplo, c:/simon
