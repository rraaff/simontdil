Instalación de Software de Base (Instrucciones para Linux Debian)

1) Java 1.6

1.a) Instalar java 1.6 con el siguiente comando: apt-get install sun-java6-jdk

1.b) Setear el JAVA_HOME 
nano ~/.bashrc
export JAVA_HOME=/usr/lib/jvm/java-6-sun

2) Tomcat 6.0

2.a) Descomprimir el TOMCAT y moverlo a la ubicación definitiva: 
 tar -zxvf apache-tomcat-6.0.32.tar.gz
mv apache-tomcat-6.0.32/ /usr/local/tomcat

2.b) Crear el grupo para el usuario tomcat
groupadd tomcat
useradd -g tomcat -d /usr/local/tomcat tomcat
usermod -G www-data tomcat

2.c) Crear un script en /etc/init.d
nano /etc/init.d/tomcat

2.d)Configurar el siguiente contenido y salvarlo:
# Tomcat auto-start
# description: Auto-starts tomcat
# processname: tomcat
# pidfile: /var/run/tomcat.pid
export JAVA_HOME=/usr/lib/jvm/java-6-sun
case $1 in
start)
  sh /usr/local/tomcat/bin/startup.sh
  ;;
stop)  
  sh /usr/local/tomcat/bin/shutdown.sh
  ;;
restart)
  sh /usr/local/tomcat/bin/shutdown.sh
  sh /usr/local/tomcat/bin/startup.sh
  ;;
esac   
exit 0

2.e) Dar permisos de ejecución al script del punto 2.d, con el siguiente comando:
chmod 775 /etc/init.d/tomcat

2.f) Agregarlo para auto-start en arranque
update-rc.d tomcat defaults
Este script permitirá una vez terminada la configuración poder iniciar, parar o reiniciar el servicio, ejecutando los comandos: /etc/init.d/ start/stop/restart

3) MySQL

3.a) Copiar el archivo mysql-connector-java-5.1.13-bin.jar al directorio lib/ext donde se encuentre instalado java, por ejemplo:
/usr/lib/jvm/java-6-sun-1.6.0.22/jre/lib/ext/

3.b) MySQL
Si no se tiene un mysql disponible se puede instalar con el siguiente comando: apt-get install mysql-server
